### [NyayaMitra - AI Legal Companion](https://github.com/Prabhatvrma1/HackInia-Spark-7-2025-Pragati)

> An AI-powered chatbot that makes Indian legal knowledge accessible to everyone using RAG architecture and LLaMA 2.

**Tech Stack:** React.js • TypeScript • Flask • LLaMA 2 • LangChain • FAISS • RAG

**Features:**
- 🤖 AI-powered legal query system with citation-backed answers
- 📚 Covers IPC, RTI, labor laws, FIR procedures
- 🔍 Lawyer discovery and registration system
- 🛡️ Privacy-first (runs locally via Ollama)

**What I learned:**
- Building RAG systems from scratch
- Integrating LLMs with vector databases
- Full-stack TypeScript development
- Managing complex state in React applications
